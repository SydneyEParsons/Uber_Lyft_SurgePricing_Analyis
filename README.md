# Uber_Lyft_SurgePricing_Analyis

This is the term project of Lingyun Jiang and Sydney Parsons for Data Analysis 5620.

The dataset we chose to analyse is the Uber and Lyft Dataset created by Ravi Munde (https://www.kaggle.com/datasets/ravi72munde/uber-lyft-cab-prices).

*Note:* Before running the code please generate the
merged_data_cleaned.csv. To generate this file ensure you have saved the raw data
files from the repository here (https://github.com/SydneyEParsons/Uber_Lyft_SurgePricing_Analyis/tree/main) and then run Cleaning And Merging The Data.R found in the R folder here on GitHub (https://github.com/SydneyEParsons/Uber_Lyft_SurgePricing_Analyis/blob/main/R/Cleaning%20And%20Merging%20The%20Data.R).
Due to file size limitations we were unable to upload the compiled merged_data_cleaned.csv file
directly.
