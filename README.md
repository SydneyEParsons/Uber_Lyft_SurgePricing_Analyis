# Uber_Lyft_SurgePricing_Analyis

This is the term project of Lingyun Jiang and Sydney Parsons for Data Analysis 5620.

The dataset we chose to analyse is the Uber and Lyft Dataset created by Ravi Munde (https://www.kaggle.com/datasets/ravi72munde/uber-lyft-cab-prices).

*Note*: Before running the code please run Cleaning And Merging The Data.R to generate the merged_data_cleaned.csv. The raw data files required for this analysis are in the github repository, due to size limitations we were unable to upload the compiled merged_data_cleaned.csv file.
